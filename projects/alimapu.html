Gestión Alimapu v2

Sistema de gestión integral para la orquesta musical Alimapu. Permite gestionar miembros, instrumentos, ensayos, eventos y comunicaciones. Construido como monorepo con Nx.

Características

  • Gestión de miembros y roles
  • Control de inventario de instrumentos
  • Calendario de ensayos y eventos
  • Sistema de comunicaciones internas
  • Autenticación y autorización
  • API REST documentada

Tech Stack

Angular 21 NestJS 11 Prisma PostgreSQL Nx TypeScript
