Gestión de Libros

Proyecto fullstack creado para practicar y aprender el uso de IA y tecnologías modernas en programación. Sistema CRUD completo para gestión de libros con autenticación.

Características

  • CRUD completo de libros
  • Autenticación de usuarios
  • API REST con NestJS
  • Frontend con Angular
  • Base de datos PostgreSQL
  • Integración con herramientas de IA

Tech Stack

NestJS Angular PostgreSQL TypeScript
